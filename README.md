# Project Title

DataSpark-Global Electronics

# Project Overview

In This Project I aim to discover the insights and trends of Global Electronics which is a leading reailer of consumer electronics
and this dataset consists of information about customers, products, sales, stores and currency exchange rates. I did analysis with this data to 
improve their marketing strategies and improve sales forecasting.these insights will help Global Electronics business growth and represented my analysis
in PowerBI Dashboard.

# Architecture

Raw Data -> Pandas -> EDA(Exploratory Data Analysis) -> Cleaned Data -> SQL Query -> PowerBI Dashboard

# TechStack

 * Pandas
 * Mysql
 * PowerBI
 * Matplotlib
 * Seaborn
 * numpy
